# onboard-gustavo-seiji-backend

## Enviroment and Tools
- Typescript
- Node
- NPM
- Prisma
- Fastify
- Postgres
- Docker

## Steps to run and debug
1. Clone the repo
2. Install the dependencies `npm install`
3. Create the container `docker compose up -d`
4. Create the database
5. Create `.env` file from `sample.env`
6. Run the migrations `npm run migrate`
7. Run the project `npm start`
